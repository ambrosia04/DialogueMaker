## Character page
Here you start with the Talk Dialogue (which you can edit by **double-clicking** so it can start with a dialogue). <br>
If you hover over a Dialogue Node, a white dot will appear on the bottom right. <br>
If you drag out from it, you'll be able to create a new node. You can also click the Create Option.<br>
A new screen will appear asking you to input:
- Option (That being the text on the arrow): What your player character tells the NPC.
- Resulting dialogue (That being the name of the new node): What the NPC will respond to the player given that option.
- Notes : In case it also will give the character friendship points, quests, items...
